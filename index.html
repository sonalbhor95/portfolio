IMDB Sentiment Analysis

NLP pipeline classifying movie review sentiment on 50,000 labelled samples.

Logistic Regression and Multinomial Naïve Bayes both achieved 89% accuracy, outperforming Linear SVM on highly polar text data. Pipeline includes stemming, stopword removal, and TF-IDF vectorisation.

Credit Card Fraud Detection

Fraud classifier on a severely imbalanced dataset (0.172% fraud rate) using SMOTE and ADASYN.

Random Forest with oversampling achieves F1-score of 86.47%, minimising both missed fraud and blocked genuine transactions.

Python Random Forest SMOTE ADASYN
